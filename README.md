## 🚀 **Sistema Completo de Gestão para Perfumarias**

O **Aroma Data** é um sistema de gestão empresarial desenvolvido especialmente para lojas de perfumaria e cosméticos. Com uma interface moderna e intuitiva, oferece todas as ferramentas necessárias para administrar seu negócio de forma eficiente.

---

## ✨ **Funcionalidades**

### 📊 **Dashboard Interativo**
- Visão geral do negócio em tempo real
- Gráficos de vendas por período
- Alertas automáticos de estoque baixo
- Cards com principais métricas (total de vendas, receita, lucro, ticket médio)

### 👥 **Gestão de Clientes**
- Cadastro completo com CPF, telefone, email e endereço
- Busca avançada por nome, CPF ou email
- Histórico de compras por cliente
- Exclusão definitiva com verificação de dependências

### 📦 **Controle de Produtos**
- Cadastro completo com marca, volume, preços
- Controle de estoque automatizado
- Alertas visuais de estoque baixo
- Atualização de estoque em tempo real nas vendas

### 🏢 **Gestão de Fornecedores**
- Cadastro completo com CNPJ, telefone, email
- Vínculo com produtos
- Busca por nome ou CNPJ

### 🛒 **Sistema de Vendas**
- Interface intuitiva para registro rápido
- Busca de produtos em tempo real
- Seleção de clientes cadastrados ou venda avulsa
- Cálculo automático de totais
- Atualização automática do estoque
- Cancelamento de vendas com restauração de estoque

### 📈 **Relatórios Avançados**
- **Filtros inteligentes:** por período, cliente, produto, fornecedor, bairro
- **5 tipos de relatório:** vendas, clientes, produtos, fornecedores, bairros
- **Exportação:** CSV e PDF com formatação profissional
- **Estatísticas:** totais, médias, rankings

### 👤 **Gerenciamento de Usuários (Admin)**
- Controle de acesso por perfil (Admin/Usuário)
- Cadastro, edição e exclusão de usuários
- Proteção contra auto-exclusão
- Logs de atividades

## 🛠️ **Tecnologias Utilizadas**

### **Frontend**
- ⚛️ React 18
- 🎨 Tailwind CSS
- 🎭 Shadcn/UI
- 📊 Recharts (gráficos)
- 📅 Date-fns
- 🔄 React Query
- 🧭 React Router DOM
- 📄 jsPDF (geração de PDF)

### **Backend**
- 🟢 Node.js
- 🚂 Express
- 🔐 JWT (autenticação)
- 🔑 Bcrypt (hash de senhas)
- 🐘 PostgreSQL
- ⚡ Redis (cache/sessões)

### **Infraestrutura**
- ☁️ Render (frontend e backend)
- 🗄️ Neon (PostgreSQL serverless)
- 🔄 Redis Cloud
- 🐙 GitHub
